# Albert Louca — CV / Resume

A modern, responsive personal portfolio website built with vanilla HTML, CSS, and JavaScript.

## Features

- 🎨 **Premium dark theme** with purple gradient accents
- ✨ **Animated particle background** with mouse interaction
- 📱 **Fully responsive** — works on desktop, tablet, and mobile
- 🎭 **Scroll reveal animations** with staggered transitions
- 📊 **Animated counters** for key statistics
- 🧭 **Sticky navigation** with active section detection
- 📄 **SEO optimized** with proper meta tags

## Sections

1. **Hero** — Introduction with animated stats
2. **About** — Professional summary and highlight cards
3. **Experience** — Timeline layout of work history
4. **Skills** — Categorized skill pills
5. **Education** — Degrees, research, and certifications
6. **Contact** — Email, phone, LinkedIn, and GitHub links

## Deployment

This site is designed to be hosted on **GitHub Pages**. Simply push to the `main` branch and the included GitHub Actions workflow will deploy automatically.

### Manual Setup

1. Create a new repository on GitHub
2. Push this code to the `main` branch
3. Go to **Settings → Pages** and set source to **GitHub Actions**
4. The site will be live at `https://<username>.github.io/<repo-name>/`

## Tech Stack

- HTML5
- CSS3 (vanilla, no frameworks)
- JavaScript (vanilla, no dependencies)
- Google Fonts (Inter, JetBrains Mono)
